# Task 7 – Basic Sales Summary using SQLite & Python

## 📌 Objective
Use SQL inside Python to extract simple sales information and display results using print statements and a basic bar chart.

## 🛠 Tools Used
- Python
- sqlite3
- pandas
- matplotlib
- Google Colab

## 📊 What This Project Does
- Creates a small SQLite database (`sales_data.db`)
- Creates a `sales` table
- Inserts sample sales data
- Runs SQL query to calculate:
  - Total quantity sold
  - Total revenue per product
- Displays results using:
  - Printed dataframe
  - Bar chart (Revenue by Product)

## 📁 Files Included
- `Task-7.ipynb` – Notebook with full implementation
- `sales_data.db` – SQLite database file
- `sales_chart.png` (optional)

## ✅ Outcome
Learned how to:
- Connect SQLite database in Python
- Execute SQL queries
- Load SQL results into pandas
- Visualize sales data using matplotlib
